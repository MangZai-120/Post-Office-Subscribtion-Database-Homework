# 同学必看：在你的电脑上跑起来（3 步）

> 这个系统是「数据库作业」，**必须装 MySQL 才能跑**，没有捷径。
> 按下面 3 步做，大概 20 分钟搞定。

---

## 第 1 步：装 MySQL（关键，不能省）

1. 下载 MySQL 安装器：
   https://dev.mysql.com/downloads/installer/
   （选上面那个较大的 `mysql-installer-community-8.0.x.msi`，约 400MB）

2. 双击安装，类型选 **Custom**，至少装这两个：
   - `MySQL Server` → `MySQL Server 8.0`（服务器本体）
   - `MySQL Workbench`（图形管理工具，可选但推荐）

3. 一路 Next，到 **Type and Networking** 保持默认（端口 3306），Next。

4. 到 **Authentication Method** 选 **Use Legacy Authentication Method**（重要！兼容老密码格式）。

5. 到 **Accounts and Roles**，**设置 root 密码并牢牢记住**，比如 `123456`。
   账号是 `root`，密码是你刚设的。

6. 到 **Windows Service**，保持默认（服务名 MySQL80，勾选 Start at System Startup），Next。

7. **Apply** 配置 → **Execute** 等它跑完 → Finish。

8. 验证：按 `Win+R` → 输入 `services.msc` → 回车 → 列表里应该有 **MySQL80**，状态是「正在运行」。

> 装完后，命令行里 `mysql` 命令可能还是找不到（没加 PATH），**没关系，不影响系统运行**。系统用 Python 连数据库，不靠这个命令。

---

## 第 2 步：装 Python（如果还没装）

1. 下载：https://www.python.org/downloads/
2. 安装时**必须勾选最底部的 `Add Python to PATH`**（！不勾会出问题）
3. 验证：打开 cmd，输入 `python --version`，能看到版本号即可

---

## 第 3 步：启动系统

1. 把整个项目文件夹拷到你的电脑（从 GitHub 下载 zip 解压，或 git clone）。

2. **双击项目里的 `start_all.bat`**。

3. 第一次运行会弹出窗口，提示 `Enter MySQL root password:`
   → 输入**你第 1 步设的那个 MySQL 密码**（输入时不显示，正常），回车。

4. 等几秒，会自动弹出浏览器，打开登录页。

5. 用 **`admin / admin123`** 登录，就能用了。

---

## 如果还是不行，把这两样发给我

1. **后端窗口的报错**：双击 `start_all.bat` 后，会弹出一个标题含 `Backend` 或 `run_server` 的黑色窗口。如果里面有 `[ERR]` 开头的红字，**把那几行完整复制/截图发给我**。

2. **确认 MySQL 服务**：按 `Win+R` → `services.msc` → 找有没有 `MySQL80`，告诉我「有/没有」「正在运行/没运行」。

---

## 常见报错对照

| 后端窗口显示 | 原因 | 解决 |
|------|------|------|
| `无法连接 MySQL 服务 (2003)` | MySQL 没装或没启动 | 回去做第 1 步，或去 services.msc 启动 MySQL80 |
| `MySQL 用户名或密码错误 (1045)` | 密码输错了 | 删掉项目里的 `.db_pwd` 文件，重新双击 start_all.bat，输正确密码 |
| `Python not found` | 没装 Python 或没加 PATH | 回去做第 2 步，勾 Add to PATH |
| `No module named pymysql` | 缺库（一般会自动装） | 打开 cmd 跑 `pip install pymysql` |
| 浏览器空白 / 登录页打不开 | 前端没启动 | 看有没有弹出标题含 `Frontend` 的窗口 |

---

## 默认登录账号
- 用户名：`admin`
- 密码：`admin123`

有任何问题，把**后端窗口的报错文字**发过来，别发 PowerShell 里 `mysql` 命令的报错（那个没用）。
